# Prompt Examples

These prompts help users start the backup workflow in different AI coding
systems.

| File | Use with |
| --- | --- |
| `codex.md` | Codex |
| `claude-code.md` | Claude Code |
| `qwen-code.md` | Qwen Code |
| `cursor.md` | Cursor Rules |
| `grok-build-compat.md` | Grok Build only if it supports Claude Code skills |

Before using any prompt:

1. Install the matching adapter from `agent-adapters/`.
2. Keep secrets out of chat.
3. Require restore-test evidence before accepting the setup.

If a code system is not listed, use the root README as a manual runbook.
