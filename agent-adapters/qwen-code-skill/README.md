# Qwen Code Skill Adapter

This folder contains the Qwen Code Agent Skill version of Free Google Disk
Backup.

Install it as a personal Qwen Code skill:

```text
agent-adapters/qwen-code-skill/free-google-disk-backup
  -> ~/.qwen/skills/free-google-disk-backup
```

Or install it into one project:

```text
agent-adapters/qwen-code-skill/free-google-disk-backup
  -> <project>/.qwen/skills/free-google-disk-backup
```

Qwen Code discovers skills from `SKILL.md`. You can ask Qwen Code to use this
skill by name, or browse available skills with `/skills`.

Invocation:

```text
Use free-google-disk-backup.
```

Do not copy secrets into the skill folder.
